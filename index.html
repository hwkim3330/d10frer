🚀 Quick Start

1. Clone Repository

git clone https://github.com/hwkim3330/d10frer.git
cd d10frer

2. Install Dependencies

pip3 install --break-system-packages scapy pyserial netifaces
pip3 install --break-system-packages pandas matplotlib plotly

3. Configure D10 for FRER

python3 scripts/configure_d10_frer.py --ip <D10_IP>

4. Run Traffic Test

# On Device A (generator)
sudo python3 scripts/generate_frer_traffic.py --interface enp11s0

# On Device B (analyzer)
sudo python3 scripts/analyze_frer_traffic.py --interface enp15s0
